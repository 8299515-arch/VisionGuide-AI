# VisionGuide AI - Mobile (Flutter)

## 🚀 Overview
Flutter application for VisionGuide AI — an AI-powered accessibility assistant for blind and visually impaired users.

## 🧠 Core Principles
- Voice-first interface (no screen dependency)
- Full accessibility (TalkBack / VoiceOver)
- Offline-first capability
- Low-latency AI interaction

## 📱 Modules (Planned)
- AI Vision Camera
- Voice Assistant
- Navigation System
- Object Detection UI
- Emergency SOS
- AI Reader (OCR)

## 🏗 Architecture
- Flutter (Clean Architecture)
- MVVM pattern
- Riverpod / Bloc (TBD)
- REST API + WebSocket streaming

## 🔗 Backend Integration
Connects to FastAPI backend:
- /auth
- /ai/vision
- /ai/ocr
- /navigation
- /emergency
